# مستكشف التخصص — جامعة أم القرى (استبيان محلي)

موقع ثابت (HTML/CSS/JS) لتوجيه الطلاب حسب مسار الثانوي والاستبيان. لا يحتاج خادماً خلفياً.

## جلسات متعددة على نفس الرابط

- **كل جهاز أو متصفح** يحصل تلقائياً على **معرّف زائر** مخزَّن في `sessionStorage`، وبيانات الخطوات تُحفظ تحت مفتاح يعتمد على هذا المعرّف — فلا تختلط جلسة هاتفك مع جلسة جهاز صديقك عند فتح نفس الرابط.
- على **نفس الجهاز** لمستخدم آخر: اضغط **«دخول كمستخدم جديد»** في أعلى الصفحة، أو افتح الرابط مع `?new=1` (مثال: `https://اسمك.github.io/المستودع/?new=1`).

## النشر على GitHub Pages (بدون localhost)

1. أنشئ مستودعاً جديداً على GitHub وارفع محتويات هذا المجلد (يحتوي `index.html` في الجذر).
2. من المستودع: **Settings → Pages**.
3. تحت **Build and deployment**: المصدر **Deploy from a branch**، الفرع `main` (أو `master`)، المجلد **`/ (root)`**.
4. احفظ. بعد دقيقة تقريباً يظهر الرابط مثل:  
   `https://<اسم_المستخدم>.github.io/<اسم_المستودع>/`
5. افتح الرابط من **الهاتف أو أي جهاز** — يعمل مباشرة عبر HTTPS دون تشغيل خادم على جهازك.

> **ملاحظة:** المسارات في المشروع نسبية (`css/`, `js/`, `data/`)، لذلك يعمل مع GitHub Pages طالما الملفات في جذر الموقع المنشور.

## ملف `.nojekyll`

يُضاف في الجذر ليمنع Jekyll من تجاهل ملفات قد تبدو خاصة؛ المشروع لا يعتمد على Jekyll.

## الخصوصية

البيانات تبقى في متصفح الزائر (`sessionStorage` / `localStorage` لسجل «العملاء المحلي»). لا يُرسل شيء إلى خادم إلا إذا أضفت لاحقاً واجهة برمجية.

## التطوير المحلي (اختياري)

```bash
cd المسار_للمجلد
python -m http.server 8080
```

ثم افتح `http://localhost:8080`.
